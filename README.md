# 田馥甄檔案庫 · Hebe Tien Archive

收錄田馥甄（Hebe）從出道至今的影音資料，涵蓋個人作品與 S.H.E 時期，所有影片均附 YouTube / Bilibili 連結，支援搜尋與篩選。

## 收錄內容

| 分類 | 說明 |
|------|------|
| 個人 MV | 歷張專輯官方 MV、Lyric Video |
| S.H.E MV | S.H.E 時期全作品 |
| 演唱會 | 如果、一一、田調等巡演，含完整場次與精彩片段 |
| 綜藝節目 | 夢想的聲音、康熙來了、金曲獎表演等 |
| 採訪訪談 | 歷年專訪、幕後花絮、記者會 |
| 影視單曲 | 小幸運、愛了很久的朋友等 OST 與數位單曲 |
| 合唱合作 | 與吳青峰、林宥嘉、五月天等合作演出 |

## 使用方式

直接訪問網站，透過左側分類瀏覽，或使用搜尋欄查找特定影片。支援按平台（YouTube / Bilibili）篩選。

## 資料來源

所有連結來自 YouTube、Bilibili 與 Google Search，經過自動驗證確保可用。資料庫持續更新中。
